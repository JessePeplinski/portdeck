# ADR 0001: Use external provider CLIs

PortDeck will bundle only its local-discovery helper and Node.js runtime; Convex, Supabase, Wrangler, Railway, flyctl, and Netlify will resolve supported user-installed CLIs through an authoritative override, the login shell, and standard Homebrew paths. This reduces the installed app from roughly 1 GB to a hard target below 110 MiB and removes the provider dependency, signing, licensing, and supply-chain burden from every release. The tradeoff is an explicit setup step and supported-version contract for users. Provider tabs remain visible with copyable install commands, official documentation, and Refresh, while PortDeck never installs or upgrades CLIs automatically. Re-embedding provider CLIs requires a new ADR with measured bundle impact and a concrete product reason.
